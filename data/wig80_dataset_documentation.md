## Field Descriptions

| Field | Description | Data Type | Notes |
|-------|-------------|-----------|-------|
| `company_name` | Full company name | String | Official company name |
| `symbol` | Stock symbol | String | 3-4 character ticker |
| `current_price` | Current stock price | Float | In PLN per share |
| `change_percent` | Daily price change | Float | Percentage change |
| `pe_ratio` | Price-to-Earnings ratio | Float | May be null if unavailable |
| `pb_ratio` | Price-to-Book ratio | Float | May be null if unavailable |
| `trading_volume` | Shares traded | String | Formatted (e.g., "1.5M", "250K") |
| `trading_volume_obrot` | Trading value | String | In PLN millions |
| `last_update` | Last data update time | String | Time of day (CET) |
| `status` | Data collection status | String | "success" or error message |

5. **Price and Volume Data**: Direct stock page extraction
